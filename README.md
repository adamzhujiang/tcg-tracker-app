# TCG Tracker

A web application to track trading card game (TCG) collections and portfolios. Built with **Django** and designed for real time tracking of TCG Portfolios. Users can add cards to portfolios, view card prices, and manage their collections.

---

## Features

- User authentication (login/logout)
- Create and manage portfolios
- Add cards to portfolios with pricing data from TCG API
- View card details, including last price, set, game, and rarity

---

## Tech Stack

- **Backend:** Django 5.2.7
- **Frontend:** HTML, CSS (custom theme), Bootstrap 5.3.2
- **Database:** SQLite (default Django)
- **API Integration:** JustTCG API for card data and pricing
- **Other:** Django REST Framework (for API endpoints)
